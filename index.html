Project


Hybrid Account Fraud Detection Using Gradient Boosting and Graph Neural Networks


Project Overview

This project focuses on detecting fake accounts and fake followers on online platforms and social media using a hybrid machine learning approach. The system combines the power of Gradient Boosting algorithms and Graph Neural Networks (GNN) to analyze both structured data and relational user behavior patterns.


Problem Statement

Social media platforms face increasing threats from fake accounts, bot networks, and fraudulent followers. Traditional detection systems fail to analyze complex relationship-based patterns between users. The goal of this project is to develop a scalable and accurate fraud detection system that can identify malicious accounts using advanced machine learning techniques.


Technologies Used

  1. Python
  2. Machine Learning
  3. Gradient Boosting Algorithm
  4. Graph Neural Networks (GNN)
  5. Data Preprocessing Techniques
  6. Model Evaluation Metrics

Methodology

  1. Data Collection from user activity and interaction patterns
  2. Data Cleaning and Preprocessing
  3. Feature Engineering based on account behavior
  4. Applying Gradient Boosting for classification of suspicious accounts
  5. Using Graph Neural Networks to analyze user connection relationships
  6. Model tuning for improved accuracy
  7. Performance evaluation using precision, recall and accuracy metrics

Key Features

  1. Hybrid approach combining traditional ML and graph-based learning
  2. High accuracy in detecting fake accounts
  3. Behavior pattern analysis
  4. Relationship mapping between users
  5. Scalable architecture for large datasets

Outcome

The system successfully improves fake account detection accuracy by analyzing complex user behavior patterns and network connections. The hybrid model demonstrates better performance compared to traditional single-algorithm fraud detection systems.


Future Enhancements

  1. Integration with real-time social media APIs
  2. Deployment as a cloud-based detection system
  3. Enhancing scalability using distributed computing
